# SEO Coach - Nederland

AI-powered SEO coaching speciaal ontworpen voor Nederlandse kleine bedrijven. Krijg gepersonaliseerd advies voor het verbeteren van je online zichtbaarheid.

## Functies

- **Nederlandse Interface**: Volledig in het Nederlands voor lokale bedrijven
- **Bedrijfsprofiel Management**: Sla je bedrijfsgegevens veilig op
- **SEO Analyse**: Krijg concrete tips voor je website
- **Content Strategie**: Hulp bij het maken van SEO-vriendelijke content
- **Progressive Web App**: Installeer als app op je telefoon
- **Offline Functionaliteit**: Werk door zonder internetverbinding

## Voor Wie

Deze tool is speciaal gemaakt voor:

- **Kleine Bedrijven**: MKB-ondernemers in Nederland
- **Webshop Eigenaren**: E-commerce SEO optimalisatie
- **Lokale Dienstverleners**: Lokale SEO verbetering
- **Content Creators**: SEO-vriendelijke content maken
- **Marketing Teams**: SEO strategie ontwikkeling

## Hoe Te Gebruiken

1. **Bedrijfsprofiel Aanmaken**: Voer je bedrijfsgegevens in
2. **SEO Doelen Instellen**: Vertel waar je hulp bij nodig hebt
3. **Gepersonaliseerd Advies**: Krijg concrete, uitvoerbare tips
4. **Voortgang Bijhouden**: Monitor je SEO verbeteringen
5. **Mobiel Gebruik**: Installeer als app voor onderweg

## Installeren als App

1. Bezoek deze pagina op je mobiele telefoon
2. Zoek naar de "Installeren" knop in je browser
3. Voeg toe aan je startscherm
4. Geniet van de app-ervaring met offline functionaliteit

## Technische Specificaties

- **Framework**: Gradio 5.x + FastAPI
- **Database**: SQLite voor bedrijfsprofielen
- **AI Integratie**: Geavanceerde taalmodellen voor SEO advies
- **PWA Functies**: Service worker, offline caching, push notificaties
- **Prestaties**: Geoptimaliseerd voor HuggingFace Spaces

## Privacy & Veiligheid

- **Lokale Opslag**: Je bedrijfsgegevens blijven veilig opgeslagen
- **GDPR Compliant**: Voldoet aan Nederlandse privacywetgeving
- **Geen Data Verkoop**: We verkopen nooit je bedrijfsgegevens
- **Veilige Verbinding**: Alle communicatie is versleuteld

## Architectuur

- **Modus**: SEO Coach voor Nederlandse bedrijven
- **Doelgroep**: MKB-ondernemers, webshop eigenaren, lokale bedrijven
- **Deployment**: HuggingFace Spaces met PWA mogelijkheden
- **Opslag**: Lokale SQLite database voor bedrijfsprofielen
- **Beveiliging**: CSP headers, veilige omgevingsvariabelen

## Omgevingsvariabelen

De volgende omgevingsvariabelen zijn geconfigureerd:

- `APP_MODE=seo_coach` - SEO Coach modus
- `ENABLE_PWA=true` - Progressive Web App functies
- `PWA_THEME_COLOR=#10b981` - Groen thema voor business gebruikers
- `DEFAULT_LANGUAGE=nl` - Nederlandse interface
- API sleutels voor AI modellen (veilig geconfigureerd)
